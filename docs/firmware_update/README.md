# ファームウェアのアップデート手順
AZ-Macroのファームウェアのアップデート手順の説明をします。  
  
  
## 対象のbinファイルのダウンロード
まずはアップデートするbinファイルをgithubからダウンロードします。  
下記リンクを開き対象のbinファイルをダウンロードして下さい。  

[ファームウェア リスト リンク](/firmware/bin/)  
  


対象のファイルを選びます。  
![ファイルの選択](/images/github_bin.png)  
  
  
「Download」を押します。  
![ファイルの選択](/images/github_bin_download.png)  
  
  
「ファイルを保存する」を選択して「OK」を押します。  
![ファイルの選択](/images/github_bin_download_ok.png)  
  
  
## 設定画面からファームウェア更新

- AZ-Macroを設定モードで起動し、PCから設定画面を開きます  
  
  
- 「設定メニュー」から「ファームウェア」を押します  
  
  
- 先ほどダウンロードしたbinファイルを選択します  
![ファイルの選択](/images/firmware_file_select.png)  
  
  
- 「アップデート」を押します  
![アップデート](/images/firmware_update.png)  
  
  
- binファイルがAZ-Macroに送られます  
![ファイル送信中](/images/firmware_update_now.png)  
  
  
- 送信が完了するとAZ-Macroがファームウェアの更新作業が始まります。(1分ほどかかりますのでそのままお待ち下さい)  
![ファームウェア更新中](/images/firmware_update_now2.png)  
  
  
- 完了すると画面が切り替わりますのでページを閉じて下さい。  
![完了](/images/firmware_update_comp.png)  
  
  
## 注意点

ファームウェア更新後はプロファイルが書き換わっている事があるためペアリング設定を１度削除し、再度ペアリングしてからご使用ください。  
  
binファイルに不備があってファームウェアが更新できなかった場合でも「アップデート成功」のメッセージが出てしまいます。  
ファームウェアが更新されたかどうかは再度管理ページを開いてファームウェアのバージョンをご確認下さい。
