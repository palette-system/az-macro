# AZ-Macro ファームウェア
AZ-Macroで使用しているファームウェアソースです。<br>
こちらのソースは個人で利用する場合には自由にご利用いただいて大丈夫です。<br>
二次配布については非営利であれば著作権表示をして頂ければご利用して頂いて大丈夫です。<br>


<br><br>


ライセンス：　CC BY-NC-SA 4.0

<br><br>

## 対応商品

- AZ-Macro
- AZ-Macro (option)踏みキー


## 開発環境

- Arduino 1.8.13
- esp32 1.0.4
- ESP OTA 0.0.3
- NimBLE-Arduino 1.0.2
- ArduinoJson 6.17.2
