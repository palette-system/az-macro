# AZ-Macro ファームウェア
AZ-Macroで使用しているファームウェアソースです。


## 対応商品

- AZ-Macro
- AZ-Macro (option)踏みキー


## 開発環境

- Arduino 1.8.13
- esp32 1.0.4
- ESP OTA 0.0.3
- NimBLE-Arduino 1.0.2
- ESP32-BLE-Keyboard 0.2.2
- ArduinoJson 6.17.2
